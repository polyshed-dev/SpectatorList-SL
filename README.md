# SpectatorListRueI
Simple [EXILED](https://github.com/ExMod-Team/EXILED) plugin that displays who is spectating you when you are alive using the [RueI](https://github.com/pawslee/RueI) hint framework. This plugin solves the hint overlap problems that the other SpectatorList plugins had.
# Depencencies
- EXILED `9.14.2`: https://github.com/ExMod-Team/EXILED
- RueI hint framework `3.1.2`: https://github.com/pawslee/RueI
# How to install
1. Download the latest release .dll file.
2. Put the file in your EXILED plugins folder.
3. Restart your server to activate the plugin.
